# Source audio (not shipped in the app)

Put the **original full-length MP3s** here when you need to run `npm run music:clips`. The clip script reads only this folder; bundled tracks live in `../clips/*.m4a`.

Filenames expected by `scripts/audio/make-music-clips.mjs`:

- `amaksi-lounge-bar-233471.mp3`
- `backgroundmusicforvideos-jazz-bar-music-restaurant-whiskey-coctail-background-intro-theme-284460.mp3`
- `bransboynd-cocktail-bar-421413.mp3`
- `moonpub-slow-hotel-lounge-jazz-music-506551.mp3`
- `soulprodmusic-bar-sad-calm-chillhop-lofi-background-music-122128.mp3`

This directory is listed in `.gitignore` so large sources stay local or in your own storage.
